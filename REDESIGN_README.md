## 🎨 Design System

### Colors
```
🔵 Blue:   #4285F4 - Primary actions
🔴 Red:    #EA4335 - Danger/Lock
🟡 Yellow: #FBBC04 - Warning
🟢 Green:  #34A853 - Success/Unlock
```

### Dimensions
```
Card corners:  24dp
Card elevation: 2dp
Button height:  56-64dp
Card padding:   20-24dp
Card margins:   16-20dp
```

### Typography
```
Headers: 18-24sp, Bold
Body:    14sp, Regular
Buttons: 14-16sp, Bold
```
